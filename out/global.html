Adds a movie to a specific user's favorites.
Parameters:
Name Type Description
'/users/:user/Movies/:MovieID' String The users endpoint with a specific user and movie ID.
jwt Object The bearer json web token passed into the HTTP request from the client.
Source:
  • index.js, line 309
Returns:

Type
Object
Example

      method: 'post',

})

Create a user('/users', user) → {JSON}

Create a user in the database. No json web token necessary, as they are a new user and will have a token assigned to them once created.
Parameters:
Name Type Description
'/users' String The users endpoint requested by the client.
user JSON The user json object containing name, username, password, email, and birthday.
Source:
  • index.js, line 234
Returns:
JSON object of the new user containing the new user's name, username, hashed password, email, and birthday.
Type
JSON
Example
axios({
    method: 'post',
    url: 'https://my-flix-api-evanoff.herokuapp.com/client/users', 
    {
      "name": "Bob Smith",
      "username": "bob123",
      "password": "smith123",
      "email": "bob@gmail.com",
      "birth_date": "01-01-1990"
    }

})

Update a user('/users/:username', jwt, user) → {JSON}

Update a user's information in the database. If trying to update a username that isn't their, a validation error will be thrown.
Parameters:
Name Type Description
'/users/:username' String The users endpoint and specific username requested by the client.
jwt Object The bearer json web token passed into the HTTP request from the client.
user JSON The user json object containing the updated name, username, password, email, and/or birthday.
Source:
  • index.js, line 364
Returns:
JSON object containing the updated name, username, hashed password, email, and/or birthday for the user.
Type
JSON
Example
axios({
      method: 'put',
      url: 'https://my-flix-api-evanoff.herokuapp.com/client/users/bob123',
      headers: { 'Authorization': `Bearer ${token}` },
      data: {
        "name" : "Bob Smith",
        "username": "bob123",
        password: "smith123",
        "email": "bobsmith@gmail.com",
        birth_date: "01-01-1990"
      }

})
